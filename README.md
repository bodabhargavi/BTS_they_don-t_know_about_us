# BTS - They Don't Know 'Bout Us | Lyric Visualizer 💜

A simple BTS-inspired lyric visualizer built using **HTML, CSS, and JavaScript**.

## ✨ Features

* Responsive UI
* Clean purple-themed design
* Glowing lyric animation synced with the song's intro
* Minimal and beginner-friendly code
* Lightweight frontend project

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 🚀 Getting Started

2. Open `index.html` in your browser.
3. Enjoy the visualizer!

## 📚 What I Learned

* Structuring a responsive webpage
* Working with JavaScript DOM manipulation
* Creating timed UI animations
* Improving UI with CSS transitions and effects

## 💜 Inspiration

This project was inspired by the song **"They Don't Know 'Bout Us" by BTS** and by a lyric visualization concept I came across on Instagram. I rebuilt the interface with my own simplified design and implementation as a frontend learning project.
